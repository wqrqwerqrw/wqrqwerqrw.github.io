WSL 下 vim 的系统间复制
2019年3月28日 凌晨
69 字 1 分钟
  • WSL 通过 clip.exe 可以与 windows 的剪切板连接
  • 通过指令
:w !clip.exe
  • 通过 vim 的 nmap 可以使用快捷键快速操作
  • 在~/.vimrc中添加
namp <c-c> :w !clip.exe
  • 也可以使用同样的方式仅复制一部分内容
环境
linux wsl
